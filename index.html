Course GIT - MERN STACK

Curso 1: Entorno de trabajo optimizado para React y Control de versiones: Visual Code, Git, Github, EsLint Presentación del enunciado del proyecto. Instalar Git y Visual Code. Configuración local y global. Crear un repositorio y conectarlo con GitHub. Crear un issue en github. Convención de branches: master, dev, issues. Desde consola y VC: agregar y quitar cambios, hacer un commit (convención de nombre. ej: número de issue, descripción), deshacer un commit local y remoto, tags, fusionar branches, manejar conflictos, fetch, pull, push, forzar un push. Historial de cambios (logs), staging, hacer rebase interactivo para actualizar un branch con lo último en dev. Pick and squash. Cherry-pick. Crear merge-request, revisar código de un compañero y aceptar merge request. Instalar EsLint y dar ejemplos simples con javascript (mostrar sumas, restas, frases en el DOM).
